The Security Incident Mapping (SIM) System is a web-based application designed to help students, staff and members of the Adekunle Ajasin University community report and monitor security incidents quickly and efficiently. The system promotes timely reporting, improved communication and a safer campus environment.

Welcome to the Security Incident Mapping (SIM) System. This platform allows students, staff and members of the AAUA community to report and view security incidents.
